Detailed Description

Class that encapsulates the following information of a particular node:

  • Input data/ports
  • Output data/ports
  • Any additional node options, as defined by e.g. dropdowns in the view.

Sets the data for an output port. All data is encapsulated by a custom type. For example, IntegerDataType encapsulated integers. This function allows you to directly output an integer as an IntegerData without doing all the intermediate steps.

Template Parameters
TThe type of the data that the output should be of. In the example, this would be IntegerData.
UThe type of the data that is encapsulated by the output port. This does not need to be set explicitly and can be inferred by the compiler. In the example, this would be int.
Parameters
nameThe name of the output port.
dataThe data to set.
