Detailed Description

template<class T, class U>
class nitro::FlexibleData< T, U >

A flexible data type that can contain any type.

Template Parameters
TType of the data it contains. This can be for example, int, double, cv::Mat or std::shared_ptr<cv::Mat>.
UType of the derived class. Used to ensure each derived class gets its own conversion map, regardless of the type of T. For example, if two data types both encapsulate an int, but do different, this type U is needed to differentiate between the two.
