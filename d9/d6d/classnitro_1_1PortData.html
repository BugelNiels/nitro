Detailed Description

Represents data of a single port belonging to a node. It contains information about the name of the port, the data type that this port has and the data itself that is at this port. The name of a port is unique amongst its category. That is, all input ports have unique names and all unique output ports have unique names. However, an output port can have a name that already exists in the input ports.
